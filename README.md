- **前端**：純 HTML/CSS/JS（單一檔案，無需 build）
- **後端**：Google Apps Script（免費）
- **資料庫**：Google Sheets（免費）
- **託管**：GitHub Pages（免費）

所有資料存在你自己的 Google 帳號，我們不保留任何副本。
