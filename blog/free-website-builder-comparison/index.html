免費架站工具比較：GitHub Pages vs Wix vs Squarespace vs 開店平台

One Page Go 2026-04-02 8 分鐘閱讀

免費架站工具是許多小型商家和個人品牌的起步選擇。目前市面上的免費方案大致分為三類：靜態網頁託管（如 GitHub Pages）、網站建置器（如 Wix、Squarespace）和電商開店平台（如 WACA）。這三類工具的設計目的不同，適合的使用場景也不同。根據 GitHub 官方文件，GitHub Pages 提供每個儲存庫 1GB 的儲存空間和每月 100GB 的流量額度，完全免費且支援自訂網域和 HTTPS。以下逐一比較各工具的免費方案限制與適用情境。

GitHub Pages：免費靜態網頁託管

GitHub Pages 是 GitHub 提供的免費靜態網站託管服務，根據 GitHub 官方文件，其主要規格為：儲存庫大小建議上限 1GB、發布後網站大小上限 1GB、每月頻寬軟上限 100GB、每小時最多 10 次建置。

GitHub Pages 的優勢在於完全免費、無廣告、支援自訂網域和自動 HTTPS，且沒有平台鎖定的問題。但它只能託管靜態檔案（HTML、CSS、JavaScript），不提供後端程式執行環境或資料庫。

如何讓 GitHub Pages 具備後端功能？

搭配 Google Apps Script 和 Google Sheets，可以為靜態網站加上後端處理和資料儲存能力。Google Apps Script 免費帳號的主要額度為：每日 HTTP 請求 20,000 次、單次腳本執行時間上限 6 分鐘、每日觸發器執行總時間 90 分鐘。Google Sheets 單一試算表支援最多 1,000 萬個儲存格。

以典型的小型商家使用情境估算：假設每天 200 筆訂單，每筆訂單觸發 2 次 API 呼叫（寫入訂單 + 發送通知），每日約 400 次請求，遠低於 20,000 次的免費額度。Google Sheets 以每筆訂單 10 個欄位計算，一張工作表可容納約 100 萬筆訂單紀錄。這就是 One Page Go 採用的三層免費架構。

Wix：視覺化網站建置器

Wix 是全球最大的網站建置器之一，以拖拉式編輯器著稱，非技術人員也能自行建立網站。

免費版限制

  • 500MB 儲存空間
  • 網頁上方會顯示 Wix 廣告橫幅
  • 網址只能使用 username.wixsite.com 格式，無法綁定自訂網域
  • 不支援電商功能（無法上架商品或接受付款）

付費版價格

Wix 的付費方案以美元計價（年繳價格）：Light US$17/月、Core US$29/月、Business US$39/月、Business Elite US$159/月。以 Core 方案為例，年繳價格約為每年 US$348（約 NT$11,136），提供 50GB 儲存空間、自訂網域和基本電商功能。

Squarespace：設計導向建置器

Squarespace 以高品質設計模板著稱，適合重視視覺形象的品牌。與 Wix 不同的是，Squarespace 沒有免費方案，僅提供 14 天免費試用。

付費方案

  • Basic：US$16/月（年繳），電商交易收 3% 手續費
  • Core：US$23/月（年繳），免交易手續費
  • Plus：US$39/月（年繳），進階電商功能
  • Advanced：US$99/月（年繳），完整企業功能

Squarespace 的 Basic 方案兩年費用約 US$384（約 NT$12,288），且電商交易另收 3%。Core 方案兩年約 US$552（約 NT$17,664）。

WACA：台灣在地開店平台

WACA 是台灣本土的電商開店平台，主打不抽成、不綁約的收費模式。

入門創業版（免費）

  • 永久免費
  • 限 30 件商品上架、10 個分類、3 組優惠券
  • 全方案不抽成（金流手續費由第三方收取）
  • 支援信用卡（手續費 3.5%）和 ATM 虛擬帳號（每筆 NT$20）

付費方案

商務成長版 NT$3,099/月（年繳）支援 2,000 件商品和會員管理。高階領航版 NT$5,399/月（年繳）支援 6,000 件商品、會員分級和多幣別。全通路旗艦版 NT$8,099/月（年繳）支援 10,000 件商品和 POS 整合。

隱藏成本提醒

選擇架站工具時，除了方案月費外，還有幾項容易忽略的成本：

  • 網域費用：自訂網域每年約 NT$300-800（.tw）或 US$10-15（.com）。GitHub Pages 免費支援自訂網域，但 Wix 免費版不支援。
  • 金流手續費：不論使用哪個平台，信用卡金流通常收取 2%-3.5% 手續費，這是支付服務商（如綠界、藍新）的費用，非平台費用。
  • 轉換成本：從一個平台搬到另一個平台需要時間和精力。某些平台的資料匯出功能有限，搬家時可能會遺失歷史訂單或會員資料。
  • 學習成本：GitHub Pages 需要基本的 HTML 知識，Wix 和 WACA 的視覺化編輯器學習曲線較低但功能也受限於平台提供的模組。

常見問題

GitHub Pages 可以拿來開店嗎？
GitHub Pages 本身只能託管靜態網頁（HTML、CSS、JavaScript），不提供後端或資料庫功能。但透過搭配 Google Apps Script（免費後端）和 Google Sheets（免費資料庫），可以實現商品展示、訂單處理和 Email 通知等完整的電商功能。One Page Go 正是採用這種三層免費架構來為小型商家建置網頁應用。
Wix 免費版和付費版差在哪裡？
Wix 免費版提供 500MB 儲存空間，但會在網頁上顯示 Wix 廣告橫幅、無法綁定自訂網域（只能用 username.wixsite.com），且不支援電商功能。付費版（Core 方案 US$29/月起）移除廣告、支援自訂網域、提供 50GB 儲存空間和電商功能。
Google Sheets 當資料庫夠用嗎？
Google Sheets 單一試算表支援最多 1,000 萬個儲存格。以典型的訂單記錄（每筆約 10 個欄位）計算，一張工作表可容納約 100 萬筆訂單。搭配 Google Apps Script 每日 20,000 次 HTTP 請求的額度，對於絕大多數小型商家的流量來說綽綽有餘。
